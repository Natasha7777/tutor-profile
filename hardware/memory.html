Память


    Компьютер использует три вида памяти — оперативную, постоянную, внешнюю.
    Оперативная память предназначена для хранения переменной информации, допускает изменение своего содержимого в ходе выполнения процессором вычислительных операций. Она обеспечивает режимы записи, считывания и хранения информации. В любой момент времени доступ может осуществляться к произвольно выбранной ячейке, поэтому оперативную память называют "память произвольного доступа" — RAM (Random Access Memory).
    Постоянная память предназначена для хранения информации, которая не должна меняться в ходе выполнения процессором программы. Ее называют ROM (Read Only Memory) — память только для чтения. Она обеспечивает только режимы считывания и хранения. Постоянная память энергонезависима, сохраняет информацию и при отключенном питании компьютера.
    По способу программирования (записи информации) микросхемы постоянной памяти делятся на программируемые изготовителем масочные (ROM), однократно программируемые пользователем (Programmable ROM — PROM), многократно программируемые пользователем (Erasable PROM — EPROM). К EPROM относятся микросхемы флэш-памяти (flash memory).
    Внешняя память реализована на магнитных или оптических носителях.
